# プロジェクト: キャリアロードマップ生成ツール

## 概要

ユーザーがやりたいことを入力すると、必要なスキル・ステップをJSON形式で返す生成AIアプリ。自己批判ループにより出力品質を高める。

---

## 要件

### 機能要件

#### 1. ロードマップ生成
- [ ] ユーザーが「やりたいこと」をテキストで入力できる
- [ ] 入力を元に必要なスキル・ステップをJSON形式で出力する
- [ ] 出力の構造:

```json
{
  "goal": "やりたいこと",
  "steps": [
    {
      "order": 1,
      "title": "ステップ名",
      "description": "説明",
      "skills": ["スキル1", "スキル2"],
      "duration": "目安期間"
    }
  ]
}
```

#### 2. 自己批判ループ
- [ ] 初回出力を再度AIに渡し、内容を批判・評価させる
- [ ] 批判内容を踏まえて最終的なアウトプットを再生成する

#### 3. データ保存
- [ ] 以下をDBに格納する:
  - ユーザーインプット
  - 初回出力（JSON）
  - 批判内容（テキスト）
  - 最終アウトプット（テキスト）
  - 最終アウトプット（JSON）
